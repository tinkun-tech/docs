## Getting Started

Use the navigation tabs at the top to browse each app's documentation. Each app section contains:

- **Overview** — what the app does and how to install it
- **Changelog** — version history following [Keep a Changelog](https://keepachangelog.com/)
- **Feature guides** — detailed usage documentation for specific features

## Updates

This site rebuilds automatically whenever a new version is released in any Tinkun app.
